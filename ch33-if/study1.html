조건문 - if

if(조건){실행될 소스코드}

괄호안의 조건이 참인 경우 코드를 실행단다, 조건의 결과가 참이 아니라면 소스코드가 실행되지 않는다.

if(조건){소스코드1}else{소스코드2}

else는 그렇지 않다면 으로 해석하고 이 코드는 제시한 조건이 참이라면 소스코드1을 해석하고 그렇지 않다면 소스코드2를 해석하게 된다.

if(조건1){소스코드1}else if(조건2){소스코드2}esle{소스코드3}

조건1이 참이라면 소스코드1을 실행하고 종료

조건1이 거짓이라면 조건2를 게산하고 참이라면 소스코드2를 실행하고 종료

조건1이 거짓이고 조건2도 거짓이라면 마지막 else구문을 실행하고 종료

else if는 그렇지 않다면 만약에~ 라고 해석하고 각 조건이 한번이라도 참이 나오면 바로 옆 중괄호 코드를 실행하고 그 이후의 조건은 비교하지 않고 종료하는 구조
